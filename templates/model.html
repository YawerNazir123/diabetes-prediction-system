Model & Dataset Details

Machine Learning Model

A supervised machine learning classification model trained on medical diagnostic data to classify patients as diabetic or non-diabetic.

Input Features

  • Pregnancies
  • Glucose Level
  • Blood Pressure
  • Skin Thickness
  • Insulin
  • BMI
  • Diabetes Pedigree Function
  • Age

Output

Binary classification indicating whether the patient is Diabetic or Non-Diabetic.
